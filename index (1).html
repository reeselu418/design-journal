      🔗 Connect to GitHub

      This lets your articles save to GitHub and sync across all your devices. It takes about 5 minutes to set up.

      Step 1 — Create a GitHub account (if needed)

      Go to github.com/signup and create a free account.

      Step 2 — Create a new repository

      Go to github.com/new, name it design-journal, set it to Public, click Create repository.

      Step 3 — Upload your files

      In your new repo, click uploading an existing file, drag in both index.html and articles.json, then click Commit changes.

      Step 4 — Enable GitHub Pages

      Go to your repo → Settings → Pages → Source: Deploy from a branch → Branch: main → Save. Your site will be live at yourusername.github.io/design-journal

      Step 5 — Create a Personal Access Token

      Go to github.com/settings/tokens/new → Note: design-journal → Expiration: No expiration → Check repo scope → Click Generate token → Copy the token immediately (you won't see it again).

      Step 6 — Enter your details below

      Fill in your GitHub info to connect the admin panel.

      🔒 Your token is saved only in your browser (localStorage). It is never sent anywhere except GitHub's official API.
